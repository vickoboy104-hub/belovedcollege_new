# Report printing layout

## Modern report

- Paper: A4 portrait
- Output: one page
- Page margin: 5mm
- Order: school header, student summary, subject table, development ratings, remarks, approval and signature details
- Information panels appear below the subject table rather than beside it
- Subject-row density reduces automatically for larger result sets

## Classic report

- Paper: A4 portrait
- Output: one page
- Page margin: 5mm
- Sheet fills the available 200mm × 287mm print area
- Subject column remains narrower so assessment columns retain readable spacing

Browser print settings should use A4, portrait orientation, 100% scale, disabled browser headers/footers, and enabled background graphics.
